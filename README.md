# Student Record Management System

A console-based Student Record Management System built in C++ using Object-Oriented Programming concepts and file handling.

## Features
- Add new student records
- Display all students
- Search student by Roll Number
- Delete student record
- Update student details

## Concepts Used
- Classes and Objects (OOPs)
- File Handling (fstream, ifstream, ofstream)
- Functions
- Loops and Conditionals

## How to Run
```bash
g++ studentRec.cpp -o studentRec
./studentRec
```

## Technologies
- Language: C++
- IDE: VS Code
